
# 📊 Dashboard UI – Responsive Admin Panel
A modern and visually engaging **Admin Dashboard UI** created entirely using **HTML & CSS**. This project focuses on clean layout design, component structuring, and dashboard visualization—perfect for showcasing UI development skills without relying on JavaScript or frameworks.

**Author:** Mathavan Babu

---

## 📘 Project Overview
This manually crafted dashboard aims to simulate a professional admin panel experience. It contains multiple interface components that mimic real‑world applications used for analytics, HR dashboards, product management, and business monitoring.

This project demonstrates:
- Designing complex layouts with pure HTML & CSS
- Building reusable UI components
- Structuring dashboards with grid and flexbox
- Use of clean color palettes and spacing

The UI is inspired by modern SaaS dashboards and provides an excellent foundation for frontend learning and portfolio presentation.

## 🚀 How to Run the Project
1. Download or clone the project
2. Open the folder in your file explorer
3. Double-click on **index.html**
4. The dashboard will open in your browser automatically

No server required. No JavaScript required. No dependencies.
